# 스마트 어항
### 프로젝트 소개
원격 어항 관리 시스템을 구축하여 어항 내의 환경을 원격으로 모니터링하고 제어함으로써 어류의 건강과 환경을 유지한다.

### 참여자
202144072 - 이현종

202144105 - 안태현

202044084 - 박상우

202144081 - 장혜진

202144104 - 이소은

### 개발 환경
* 파이썬
* 안드로이드 스튜디오
* 아두이노
* 라즈베리파이

### 주요 기능
- [ ] 수조 상태 모니터링
- [ ] 수질 및 수위 측정
- [ ] 조명 제어
- [ ] 천장 자동 개폐
- [ ] 먹이 자동 급여
