# 스마트 어항
### 프로젝트 소개
스마트 어항을 제작하여 어항 내의 환경을 원격으로 모니터링하고 제어함으로써 어류의 건강과 환경을 유지한다.

### 개발 환경
* 파이썬
* 안드로이드 스튜디오
* 아두이노
* 라즈베리파이

### 주요 기능
- [ ] 수조 상태 모니터링
- [ ] 수질 및 수위 측정
- [ ] 조명 제어
- [ ] 천장 자동 개폐
- [x] 먹이 자동 급여
